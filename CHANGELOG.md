# Changelog

## 1.1
- Added language menu with English, German, Spanish, Korean, and Simplified Chinese.
- Added i18n translation table and UTF-8 text rendering via SDL2_ttf.
- Implemented font fallback for Latin/Korean/Simplified Chinese to avoid missing glyphs.
- Improved menu/input behavior (debounce, key prompt order, SPACE-only intro).
- Updated save/config and hiscores handling for Windows layout.
- Updated rendering fixes for OpenGL UI text and menu stability.
